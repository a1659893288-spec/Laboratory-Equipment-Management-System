# 実験室設備借用管理システム (Laboratory Equipment Management System)

## プロジェクトの概要
本プロジェクトは、大学や研究機関の実験室における備品管理をデジタル化し、効率的な運用を支援するために開発したWebシステムです。パソコンや精密測定器といった限られたリソースの貸出、返却、および予約状況をリアルタイムで一括管理することで、手動管理によるミスを防ぎ、備品利用の透明性を高めることを目的としています。

## システムの特徴と機能
このシステムの最大の特徴は、「学生」「教員」「管理者」という3つの異なるユーザーロールに対応している点です。それぞれの権限に応じた機能を提供しており、学生は備品の検索と予約、教員は承認フローの確認、管理者は備品の登録や全ユーザーの利用状況の監視が可能です。これにより、煩雑な借用プロセスを一つのプラットフォーム上で完結させ、スムーズな研究室運営を実現しました。

## 技術スタックと選定理由
バックエンドには信頼性の高い Java と Spring Boot 框架を採用し、MVCモデル（Spring MVC）に基づいた堅牢なアーキテクチャを構築しています。データベース操作には MyBatis を使用して SQL 制御の柔軟性を確保し、MySQL を通じて大量の備品・予約データを効率的に管理しています。また、フロントエンドの表示には JSP を活用し、直感的な操作が可能なユーザーインターフェースを提供しています。
